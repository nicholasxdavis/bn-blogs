Introducing Stellanova Vibe Tools

Nov 5, 2025 by Blacnova

Today, we’re releasing a major update that makes Stellanova Vibe Tools the ultimate companion for developers and designers. This comprehensive Chrome extension is packed with a powerful suite of tools for scraping, analyzing, and managing web assets, fonts, colors, and more—all from a single, streamlined interface.

A Unified Developer Toolset

Vibe Tools consolidates 9 essential developer utilities into one dropdown menu. Instead of juggling multiple extensions, you now have a single, focused toolset. This includes a Skin Scraper for saving pages as MHTML, an Asset Scraper to extract all images and scripts, and analyzers for CSS, SEO, and Tech Stacks.

We've also built-in interactive tools like FontGrab, ColorPick, and a Screen Ruler to make design analysis effortless.

Knowledge Bubbles: Your Project's Memory

Knowledge Bubbles are a powerful new feature for storing and organizing structured project information. Save project names, descriptions, and asset URLs (including direct image/video uploads) in a format optimized for AI model consumption.

You can drag and drop saved bubbles into any input field or chat interface, providing comprehensive context to AI assistants in seconds. It's the perfect way to keep all your project details in one reusable place.

Reusable Prompts for AI Workflows

Alongside Knowledge Bubbles, we're introducing Prompts. This feature allows you to save, organize, and quickly access commonly used text templates for AI interactions. We've included 6 built-in templates for common tasks like code review, debugging, and refactoring, and you can create and save your own.

Like Knowledge Bubbles, Prompts can be dragged directly into input fields, streamlining your entire AI-assisted development process.

A Seamless Analysis Workflow

The first thing you'll notice when you run the Skin Scraper is our new documentation generator. It saves the current page not just as an MHTML file, but also generates a comprehensive UI Design Documentation in Markdown. This file details the color palette, typography, layout, and components, giving you an instant, detailed overview of any site's design system.
